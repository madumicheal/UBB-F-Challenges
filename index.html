Welcome to Scrolling Nav

A landing page template freshly redesigned for bootstrap 4

About this page

This is a great place to talk about your webpage. This template is purposefully unstyled so
you can use it as a boilerplate or starting point for your own landing page desings! This
template features:

  • clickable nav links that smooth scroll to page sections
  • responsive behaviour when clicking nav links perfect for one website
  • Bootstrap's scrollspy feature which highlights which section of the page you're on in the navbar
  • Minimal custom CSS so you are free to explore your own unique design options
